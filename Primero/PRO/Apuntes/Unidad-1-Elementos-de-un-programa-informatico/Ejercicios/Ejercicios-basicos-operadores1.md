# Ejercicios de Operadores en Java

- Si a, b y c son variables enteras con valores __a=8, b=3, c=-5__, determina el valor de las siguientes expresiones aritméticas:
    -  a + b + c
    -  2 * b + 3 * (a – c)
    - a / b
    - a % b
    - a / c
    - a % c
    - a * b / c
    - a * (b / c)
    - (a * c) % b
    - a * (c % b)
    - (3 * a – 2 * b) % (2 * a – c)
    - 2 * ( a / 5 + (4 - b * 3)) % (a + c - 2)                      
    - (a - 3 * b) % (c + 2 * a) / (a - c)
    - a - b - c * 2
